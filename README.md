## Django-Digital-Resume

- 프로젝트 소개 :  온라인 이력서
- 기술 스택 : Python, Django, HTML, CSS, Bootstrap, Docker, AWS, Git, Github, SQLite
- 기능
    - 웹 서비스 API 서버 개발
    - Django Template 사용하여 화면 개발
